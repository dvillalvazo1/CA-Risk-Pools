# California-Incorporated-Cities-Towns
Raw files, scripts, and cleaned data for Debora Villalvazo's research on public risk pools and private insurers for California cities and towns. 

**California Joint Powers Insurance Authority Data**: Raw membership data from the Way Back Machine. Will help detect which cities belong to the public risk pool (California Joint Powers Insurance Authority) at different points in time. 
  Raw file: "risk_pool.txt" 
  .Rmd: "CJPIA.Rmd" 
  Output file: CJPIA.csv 
  
**Northern Caalifornia Cities Risk Pool**
  .Rmd: "Northern_CA_Cities.Rmd" 
  Output file: northern_ca_cities_rp.csv" 

**ScoreJpa**: Raw membership data from the ScoreJPA website. Will help detect which cities belong to the public risk pool (Small cities organized risk effort) at different points in time. 
  Raw file: "scorejpa.csv"
  .Rmd: "scorejpa.Rmd" 
  Output file: "score.csv"
